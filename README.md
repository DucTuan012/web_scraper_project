# Dự án Công cụ Web Scraping

Đây là dự án Python để theo dõi giá vàng và các nghiên cứu mới nhất về AI/CS.

## Chức năng

- `scrape_gold.py`: Lấy giá vàng từ btmc.vn.
- `scrape_ai_news.py`: Lấy 10 bài báo mới nhất về AI/CS từ arXiv.

## Cách chạy

1. Cài đặt thư viện: `pip install -r requirements.txt`
2. Chạy file: `python scrape_gold.py` hoặc `python scrape_ai_news.py`
